# DSList Project - Java Spring Intensive

## Overview

DSList is a Java Spring project developed during the **Java Spring Intensive** course by DevSuperior. This project focuses on building a RESTful API to manage game lists and their associated games, covering key concepts like domain modeling, database configuration, and API endpoints.

### `WebConfig.java`
```java
@Configuration
public class WebConfig {

    @Value("${cors.origins}")
    private String corsOrigins;

    @Bean
    public WebMvcConfigurer corsConfigurer() {
        return new WebMvcConfigurer() {
            @Override
            public void addCorsMappings(CorsRegistry registry) {
                registry.addMapping("/**").allowedMethods("*").allowedOrigins(corsOrigins);
            }
        };
    }
}
```

### `GameRepository.java`
```java
@Query(nativeQuery = true, value = """
    SELECT tb_game.id, tb_game.title, tb_game.game_year AS `year`, tb_game.img_url AS imgUrl,
    tb_game.short_description AS shortDescription, tb_belonging.position
    FROM tb_game
    INNER JOIN tb_belonging ON tb_game.id = tb_belonging.game_id
    WHERE tb_belonging.list_id = :listId
    ORDER BY tb_belonging.position
    """)
List<GameMinProjection> searchByList(Long listId);
```

### `GameListRepository.java`
```java
@Modifying
@Query(nativeQuery = true, value = "UPDATE tb_belonging SET position = :newPosition WHERE list_id = :listId AND game_id = :gameId")
void updateBelongingPosition(Long listId, Long gameId, Integer newPosition);
```
